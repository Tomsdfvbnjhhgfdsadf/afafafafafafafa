перед этим `apt update`
установить раст: `curl https://sh.rustup.rs -sSf | sh; source "$HOME/.cargo/env"; apt install pkg-config`
установить мэйк и либы: `sudo apt install make; sudo apt-get install libpcre3-dev; apt-get install libssl-dev; apt-get install gcc`

для того чтобы скомпилировать программу нужно прописать: `cargo build --release`
для того чтобы скомпилировать vanitygen(генерация кошелька): `cd vanitygen-plusplus; make`

запустить софт: `cargo run --release`

база данных находится в `src/db/db.sqlite`
чтобы забрать все данные оттуда: `select * from accounts`
чтобы удалить все данные оттуда: `delete from accounts`

настройки находятся в `config.toml` - все пункты там описаны

если выпадает ошибка: "insufficient funds for gas * price + value" - блок в котором отправлены матики/юсдт еще не обработан. софт подождет 1 секунду и попробует снова. можно не обращать внимания.
